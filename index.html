Software

Carbon

Core runtime built on Paper with extra plugin lifecycle management and admin tooling.

Celsium

CarbonMC's proxy. Works with velocity and common proxy setups for large networks.

Developer API

Extends Paper's API so plugin authors can add hot-reload hooks and state preservation helpers.

A diverse plugin ecosystem

CarbonMC supports existing Paper plugins and adds new admin-control plugins that allow instant installation, rollback, and safe sandboxing, so you'll never miss out!
